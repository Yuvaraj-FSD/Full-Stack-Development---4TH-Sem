01 Arrow Functions

Arrow functions are a shorter way to write functions. They also don't have their own this.

// Traditional function
function double(n) { return n * 2; }

// Arrow function — same thing, shorter
const double = n => n * 2;

// Arrow with filter + map
const evens = nums.filter(n => n % 2 === 0);
const doubled = evens.map(n => n * 2);

02 Destructuring

Destructuring lets you unpack values from arrays or properties from objects into separate variables.

// Object destructuring
const { name, age, city } = person;

// Array destructuring
const [first, second, ...rest] = myArray;

// With default values
const { role = "guest" } = user;

03 Template Literals

Template literals use backticks and ${ } to embed expressions directly inside strings.

// Old way (messy)
"Hello, " + name + "! You like " + subject + ".";

// Template literal (clean)
`Hello, ${name}! You like ${subject}.`

// Multi-line without \n
`Line 1
 Line 2`

04 Promises

A Promise represents an operation that hasn't completed yet. It can resolve (success) or reject (failure).

Click the button to try...
const flipCoin = new Promise((resolve, reject) => {
  const heads = Math.random() > 0.5;
  if (heads) {
    resolve("Heads! Promise resolved ✓");
  } else {
    reject("Tails! Promise rejected ✗");
  }
});

flipCoin
  .then(result => console.log(result))
  .catch(err => console.error(err));

05 Async / Await

Async/await is syntactic sugar over Promises — it makes asynchronous code look and feel synchronous.

Click to fetch a joke...
async function fetchJoke() {
  try {
    const res = await fetch("https://official-joke-api.appspot.com/jokes/random");
    const data = await res.json();
    const { setup, punchline } = data; // destructuring too!
    return `${setup} ... ${punchline}`;
  } catch (err) {
    return "Oops, couldn't fetch a joke.";
  }
}

06 Modules

ES6 modules let you split your code into separate files using import and export.

// mathUtils.js — a module
export const add = (a, b) => a + b;
export const subtract = (a, b) => a - b;
export const multiply = (a, b) => a * b;
export const divide = (a, b) => b !== 0 ? a / b : "Can't divide by zero";

// main.js — importing from that module
import { add, subtract, multiply, divide } from "./mathUtils.js";
